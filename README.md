# Moodify
#### A simple and aesthetic mood tracking website that helps users record how they feel, reflect on their emotions and view their mood history over time.

#### By **Esther Kiarie**

## Description
Moodify allows users to track their emotions, write short notes on what caused them to feel as they do, and review their mood history. It was designed to feel emotionally comforting, using soft colours, playful fonts and rounded cards to create a welcoming experience.
https://k3arye.github.io/moodify-finalproject/

## Setup/Installation Requirements
No special installation is required to run this project.

### Option 1: Open directly
1. Download or clone the repository
2. Open the folder
3. Double-click `index.html`
4. The website will open in your browser

### Option 2: Using VS Code (recommended)
1. Open the project in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html`
4. Click **"Open with Live Server"**

## Challenges Faced
Some challenges during development included:
* Keeping the header responsive across devices
* Centering the brand name while maintaining navbar alignment
* Preventing layout overlap on smaller screens
* Styling pages consistently

These challenges were solved using:
* Flexbox
* Media queries
* Responsive spacing techniques
